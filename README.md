# Todo
School task: React Native / Typescript

Todo list app enables user to create list of tasks. Data is saved into phone’s memory. User can also mark task as done/undone


## What can it do?

- Display list of tasks
- User can add new task
- User can delete tasks
- Task can be marked as done/undone by pressing a row/line on list
- Data is save into phone’s memory (AsyncStorage)
- App code is divided into (sub)components
